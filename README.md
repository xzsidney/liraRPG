# Lira RPG - Mundo das Trevas (World of Darkness)

Este é um projeto voltado para o desenvolvimento de ferramentas de suporte a campanhas e gerenciamento de personagens no universo de **Mundo das Trevas** (World of Darkness), englobando as regras mais recentes de quatro sistemas clássicos.

---

## 🌌 Sistemas Abrangidos

O projeto dá suporte e integra mecânicas dos seguintes sistemas:

1. **Vampiro: A Máscara (V5)**
   * **Foco:** Terror pessoal e política de clãs.
   * **Mecânicas Principais:** Sistema de *Fome*, *Humanidade* (máculas) e *Disciplinas*.
2. **Lobisomem: O Apocalipse (W5)**
   * **Foco:** Tragédia ambiental e fúria espiritual.
   * **Mecânicas Principais:** Sistema de *Fúria*, as *Cinco Formas* (Crinos, Glabro, etc.) e o equilíbrio entre *Harano* e *Hauglosk*.
3. **Mago: A Ascensão (M20)**
   * **Foco:** Alteração mística da realidade e a Guerra da Ascensão.
   * **Mecânicas Principais:** As *9 Esferas*, nível de *Arete*, uso de *Quintessência* e acúmulo de *Paradoxo*.
4. **Caçador: A Revanche (H5)**
   * **Foco:** Humanos comuns lutando pela sobrevivência contra o sobrenatural.
   * **Mecânicas Principais:** *Credos*, uso de *Ímpeto* e gerenciamento de *Aflição*.
